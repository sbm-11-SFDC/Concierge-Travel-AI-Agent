# Travel Agent — Kaggle Submission (Offline Demo)

This contains an offline, reproducible Kaggle notebook demonstrating the Travel Agent. The notebook uses saved demo outputs (no external API calls required) so it will run on Kaggle without network or API keys.

Files:
- notebook.ipynb         — Jupyter notebook (offline demo + optional guidance)
- requirements.txt       — pip requirements (for local testing)
- demo_outputs/          — saved JSON/text outputs used by the notebook
- demo_outputs/images/   — place sample images here (optional)
